Helpful Linux commands:

To change the time without typing an entire string. (Couldn't find this easily on the manpages or internet.)
date -s '+1 hour'
This would advance the time by one hour, -1 hour or -1 minute or whatever probably works too.

grep -lr string *
this will search all files in the current directory and all subdirectories and return the names of files containing string, where string may be a regular expression.
